# Instalacion 

Para instalar autoforce localmente en un proyecto. 


<Tabs>
  <TabItem value="npm" label="npm">
    ``` bash
    npm install --saveDev autoforce
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn" default>
    ``` bash
    yarn add -D autoforce
    ```
  </TabItem>
</Tabs>
